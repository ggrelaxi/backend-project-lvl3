## Setup

```sh
$ git clone https://github.com/ggrelaxi/backend-project-lvl3.git
```

```sh
$ npm install
```

```sh
$ npm link
```


## For download webpage, enter url and destination folder
```sh
$ page-loader <webpack_url> --output <destination_dirname>
```
